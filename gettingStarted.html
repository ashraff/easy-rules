Prerequisite

Easy Rules requires a Java 1.5+ runtime.

Build Easy Rules

To build Easy Rules from sources, you need to have git and maven installed and set up.

Please follow these instructions :

  • $>git clone https://github.com/benas/easy-rules.git

  • $>mvn package

  • Easy Rules core jar easyrules-core-${version}.jar will be generated in the target folder

Use Easy Rules

Easy Rules is a single jar file with no dependencies. You have to simply add the jar easyrules-core-${version}.jar to your application's classpath.

If you use maven, you should add the following dependency to your pom.xml :

<dependency>
    <groupId>io.github.benas</groupId>
    <artifactId>easyrules-core</artifactId>
    <version>${version}</version>
</dependency>
